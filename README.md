# playlist-maker-android-TiniakovAndrei
## Описание Проекта
Playlist Maker - мобильное приложение для создания и управления музыкальными плейлистами.
Приложение позволяет искать треки через iTunes API, добавлять их в избранное
и создавать пользовательские плейлисты с возможностью добавления обложек.

### Основные функции:

- Поиск музыкальных треков через iTunes API

- Просмотр детальной информации о треках

- Добавление треков в избранное

- Создание пользовательских плейлистов

- Добавление обложек к плейлистам

- Сохранение истории поиска

- Хранение данных между сессиями приложения

### Системные требования:

- Android Studio Narwhal 3 Feature Drop | 2025.1.3 или выше

- Kotlin: 2.2.20 или выше

- Java: 11

### Версии SDK:

- Минимальная версия Android: API 29 (Android 10)

- Целевая версия Android: API 36

- Компиляция под: API 36

### Основные библиотеки:

- Jetpack Compose для UI

- Room для локальной базы данных

- Retrofit для сетевых запросов

- Coil для загрузки изображений

- DataStore для хранения настроек

- Kotlin Coroutines для асинхронных операций

## Сборка и запуск

1. Установите Android Studio последней версии

2. Убедитесь, что установлен Android SDK версии 36

3. Установите JDK 11 или выше

4. Настройте эмулятор Android или подключите физическое устройство с Android 10+

### Шаги сборки

1.  Клонирование проекта
```
git clone https://github.com/ButterMun/playlist-maker-android-TiniakovAndrei
cd playlistmaker
```

2. Запустите Android Studio, выберите "Open" и укажите папку проекта

3.  Если автоматическая синхронизация gradle не произошла, нажмите "Sync Now" в правом верхнем углу

4.  Создайте эмулятор через AVD Manager или подключите физическое устройство с включенной отладкой по USB

5.  Нажмите кнопку "Run"
