# playlist-maker-android-TiniakovAndrei
Учебный проект по созданию мобильного приложения


## Функциональность 6 спринта

- Реализован полнофункциональный поиск треков

- Добавлена ViewModel для управления состоянием экрана поиска

- Настроены состояния поиска: начальное, поиск, успех, ошибка

- Интеграция UI с данными через StateFlow и коллекцию состояний

- Отображение списка найденных треков с информацией о треке
